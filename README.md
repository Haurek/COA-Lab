# 计算机组成原理课程lab

## Lab1

### 实验目的

- 熟悉QtSPIM模拟器；
- 熟悉编译器、汇编器和链接器；
- 熟悉MIPS体系结构的计算，包括：

  - MIPS的数据表示；
  - 熟悉MIPS指令格式和寻址方式
  - 熟悉MIPS汇编语言
  - 熟悉MIPS的各种机器代码表示，包括
    - 选择结构；
    - 循环结构；
    - 过程调用：调用与返回、栈、调用约定等；
    - 系统调用。
